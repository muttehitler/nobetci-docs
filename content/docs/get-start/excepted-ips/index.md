Excepted IPs that are't ban

## Cli

- You can manage excepted IPs with cli:

### List IPs

```sh
nobetci cli excepted_ip list
```

### Add IP

```sh
nobetci cli excepted_ip add --ip 'your_ip'
```

### Delete IP

```sh
nobetci cli excepted_ip delete --ip 'your_ip'
```
